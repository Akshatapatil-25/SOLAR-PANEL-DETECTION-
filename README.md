# SOLAR-PANEL-DETECTION-
AI-powered Solar Panel Detection using YOLOv8 and OpenCV for identifying and localizing solar installations from aerial and satellite imagery.
Runner-Up Project at the ISRO Hackathon hosted at VNIT Nagpur

## Overview

Solar Panel Detection using YOLOv8 is a Computer Vision and Deep Learning project that automatically detects and localizes solar panels from aerial and satellite imagery. The system leverages the YOLOv8 object detection framework to identify solar installations with high accuracy, generate bounding boxes, and provide confidence scores for each detection.

The project aims to support renewable energy monitoring, solar infrastructure mapping, and sustainable development by reducing the need for manual surveying and analysis.

## Features

* Automatic solar panel detection from aerial and satellite images
* Bounding box localization of detected panels
* Confidence score generation for each prediction
* High-speed and accurate object detection using YOLOv8
* Scalable solution for large geographic regions
* Potential integration with GIS and drone imagery systems
* Solar panel coverage area estimation
